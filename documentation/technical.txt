  + Node Type: Standard_DS3_v2
  + Driver Node: Standard_DS3_v2

  + Description: Customer and orders data
* **Target Data Details**:

  + Location: `bronzezone.data.customer_raw`
  + Format: Delta Lake
  + Description: Raw customer data

  + Location: `bronzezone.data.orders_raw`
  + Format: Delta Lake
  + Description: Raw orders data

  1. Read CSV data from Unity Catalog Volume
  2. Enable checkpointing and incremental loads
  3. Apply SCD Type 2 logic
  4. Write data to Bronze Layer tables
* **Data Transformations / Business Logic**:
  + Step 1: Read CSV data from Unity Catalog Volume using cloud_files
  + Step 2: Enable checkpointing and incremental loads

* **Acceptance Criteria**:

  + SCD Type 2 logic is applied correctly
  + Watermarking is applied on order_date
* **Notes / Implementation Suggestions**:

  + Enable checkpointing and incremental loads for efficient data ingestion
